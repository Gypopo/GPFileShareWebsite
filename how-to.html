What is this site for?

This site is a community marketplace which hosts custom/premade shop layouts made for EconomyShopGUI and EconomyShopGUI Premium created by community members

Uploading layouts

Uploading layouts is done using 4 simple steps:

  • Use the /eshop uploadLayout command to start the upload process of your layout
  • Enter a layout title, description and tags when asked in chat
  • Next, a GUI will open where you will be asked to select the shop layouts to upload
    • Optionally, the config.yml can be uploaded if transaction/option items, navigation bars or display lore layout has been modified and is apart of the custom layout
  • Finally, choose a upload method: Discord or Temporarily
When completed the steps above, you will need to wait till your layout gets accepted by our server.
Your layout will be denied or removed if it does not meet the guidelines listed on the Guidelines page.

Installing premade layouts

Installing shop layouts can be done using 1 command:

  • Use the /eshop installLayout [layoutID] command to start the install process of a layout

    • Replacing [layoutID] with the layout specific ID

  • After the installation proceeds, the changes can be reloaded using /sreload (Note that we recommend a server restart for layouts using a config.yml)

The install command can be located by clicking any layout on the layouts page > Get install command > Click the command to copy

Updating existing layouts

Updating existing layouts is useful for minecraft updates or simple changes/corrections in your layout

Updating the files of your layout:
  • Use the /eshop updateLayout [layoutID] command to start the update process of your layout

    • Replacing [layoutID] with the layout specific ID

Updating the title/description:
  • Login on the My Layouts page using the discord account u used while uploading the layout
  • Click one of your layouts to start editing the title/description
Adding screenshots to layouts Similarly as updating the title/description of your layout, it is possible to upload screenshots which will help visualize the layout
  • Login on the My Layouts page using the discord account u used while uploading the layout
  • Click the layout you want to upload screenshots for
  • You will now be able to click the Upload screenshot button

Note that these features are only available for layouts uploaded using Discord verification and can only be updated by the original author of the layout

Differences between upload methods:

There are two different ways to upload a layout, anonymously or using Discord authentication.

  • Temporarily upload makes it so you can upload layouts anonymously, but will be removed after 5 days
  • Discord authentication makes it possible to edit/remove your uploaded layouts at any later point when logged in thru discord on the My Layouts page
    • Layouts uploaded using Discord authentication will stay permanently available to download

Uploading/downloading layouts to/from this site requires atleast EconomyShopGUI v6.2.0 or EconomyShopGUI Premium v5.4.0.
Make sure you are up-to-date!
