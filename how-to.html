What is this site for?

This site is a community marketplace which hosts custom/premade shop layouts made for EconomyShopGUI or EconomyShopGUI Premium created by community members
For two main reasons we've made this marketplace:

  • Reason one being so other people can easily find, download and install premade shop layouts so they can start within seconds
  • Reason two being that this marketplace will give inspiration to new server owners and get them started quickly creating their own server shop
We hope that after a while this marketplace will be filled using lots of premade/custom shop layouts

How to upload, download and update premade shop layouts for EconomyShopGUI


Note that uploading/downloading layouts to/from this site requires atleast EconomyShopGUI v6.2.0 or EconomyShopGUI Premium v5.4.0.
Make sure you are up-to-date!


Uploading custom layouts
Using 4 simple steps you can upload a premade layout, for anyone to download or get inspiration from.

  • Use the /eshop uploadLayout command to start the upload process of your layout
  • Enter a layout title, description and tags when asked in chat
  • Next a GUI will open where you will be asked to select the shop layouts to upload
    • Optionally the config.yml can be uploaded if transaction/option items, navigation bars or display lore layout has been modified and is apart of this shop layout
  • Finally choose a upload method: Discord or Temporarily
When completed the steps above, you will need to wait till your layout gets accepted by our server.


Installing premade layouts
Installing premade layouts is as simple as executing 1 command

  • Use the /eshop installLayout [layoutID] command to start the install process of your layout
    • Replacing [layoutID] with the layout specific ID
  • After the installation proceeds, the changes can be reloaded using /sreload(Note that we recommend a server restart for layouts using a config.yml)
The install command can quickly be located by clicking any layout on the layouts page > Get install command > Click the command to copy


Updating existing layouts
Updating existing layouts is usefull for minecraft updates or simple changes/corrections in your layout

  • Use the /eshop updateLayout [layoutID] command to start the update process for your layout
    • Replacing [layoutID] with the layout specific ID
Note that this feature is only available for layouts uploaded using Discord verification and can only be updated by the original author of the layout

Differences between upload methods:
There are two different ways for upload layouts, anonymously or using Discord authentication.

  • Temporarily upload makes it so you can upload layouts anonymously, but will be removed after 14 days
  • Discord authentication makes it possible to edit/remove your uploaded layouts at any later point when logged in thru discord on the My Layouts page
    • Layouts uploaded using Discord authentication will stay permanently available to download
