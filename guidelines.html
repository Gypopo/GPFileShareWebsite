Resource guidelines:

Uploaded layouts should follow the following guidelines, to keep this Marketplace a safe/clean place for our community to browse and upload new layouts:

  • Layouts descriptions/names should be in Enlgish
  • Descriptive resource titles/descriptions, anyone should be able to understand what your custom shop layout is on about
    • Adding spam or filler characters is not allowed
    • Any sexual, spam, or offensive language is strictly prohibited
  • Layouts should not contain any malicious files, or content
    • Uploading malicious files may cause your IP to be banned of any of our services
  • Layouts should not be of low quality, meaning:
    • Only contains a small amount of items
    • Should contain custom content (configs/items)
    • Cannot be structured similarly to the default layout
    • Cannot be a duplicate or near duplicate of existing layouts
    • Layouts should not contain config files of the default layout, unless:
      • It is apart of a custom layout
      • The config file has been edited/contains items which are not included in the default file

If your layout does not abide the above guidelines, we have all rights to remove your layout from this site without notice
